# FormField (molecule · panel)

Label « affiche de cirque » + champ assemblés, avec aide optionnelle.

| Prop       | Type        | Rôle                    |
| ---------- | ----------- | ----------------------- |
| `label`    | `string`    | Intitulé (uppercase)    |
| `children` | `ReactNode` | Le champ (Field, etc.)  |
| `hint`     | `string`    | Texte d'aide optionnel  |

Dépendance : `Field` (passé en children).
