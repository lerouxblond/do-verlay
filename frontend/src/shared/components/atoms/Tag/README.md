# Tag (atom · partagé)

Étiquette de guilde au style « chip » doré. En lecture seule sur l'overlay ; éditable (croix de
suppression) dans le panel.

| Prop       | Type         | Défaut | Rôle                                   |
| ---------- | ------------ | ------ | -------------------------------------- |
| `label`    | `string`     | —      | Texte du tag                           |
| `onRemove` | `() => void` | —      | Si fourni → affiche la croix éditable  |

Aucune dépendance (atome).
