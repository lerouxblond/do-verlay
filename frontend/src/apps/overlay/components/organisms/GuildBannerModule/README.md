# GuildBannerModule (organism · overlay)

Étendard de guilde complet : blason réel, nom, niveau, état de recrutement et tags (≤ 5).

| Prop    | Type    | Rôle              |
| ------- | ------- | ----------------- |
| `guild` | `Guild` | Données de guilde |

Dépendances : `CardShell`, `GuildCrest`, `Tag`, `StateBadge` (via GuildCrest).
