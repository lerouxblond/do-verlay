# ProgressBar (atom · partagé)

Jauge de progression au motif harlequin (remplissage rouge Zobal + losanges or).

| Prop     | Type     | Défaut | Rôle                  |
| -------- | -------- | ------ | --------------------- |
| `value`  | `number` | —      | Valeur courante       |
| `max`    | `number` | —      | Valeur maximale       |
| `height` | `number` | `12`   | Hauteur de la jauge   |

Le pourcentage est borné à [0, 100]. Aucune dépendance (atome).
